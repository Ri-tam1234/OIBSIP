# Web Development Projects

This repository contains three beginner-friendly web development projects built using **HTML, CSS, and JavaScript**. These projects helped me strengthen my understanding of responsive layouts, styling, and JavaScript fundamentals.

## Projects

### 1. Landing Page

A responsive landing page built using HTML and CSS. The project focuses on creating a clean layout with proper spacing, typography, and responsive design.

**Features**

* Responsive layout
* Navigation bar
* Hero section
* About and Services sections
* Footer
* Modern UI

**Technologies Used**

* HTML5
* CSS3

---

### 2. Portfolio Website

A personal portfolio website to showcase my skills, projects, and contact information.

**Features**

* Responsive navigation
* Hero section
* About section
* Skills section
* Projects section
* Contact section

**Technologies Used**

* HTML5
* CSS3

---

### 3. Temperature Converter

A temperature converter that allows users to convert values between Celsius, Fahrenheit, and Kelvin using JavaScript.

**Features**

* Celsius to Fahrenheit and Kelvin
* Fahrenheit to Celsius and Kelvin
* Kelvin to Celsius and Fahrenheit
* Input validation
* Responsive design

**Technologies Used**

* HTML5
* CSS3
* JavaScript
